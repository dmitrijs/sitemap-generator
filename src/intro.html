Introduction to Sitemap Generator



How to use Sitemap Generator

  1. Click extension icon     in top right corner of chrome browser

  2. Session configuration window will open. Set your preferences then click start.

  3. Sitemap generator will launch in a separate window. You may let this window run in the background.

  4. Once the generator is done, sitemap will open in new tab and download.

A few words about permissions

When you use use the extension to crawl some domain for the first time you should see permissions prompt. These permissions are required for sitemap generation to be successful so make sure you choose Allow.

What the prompt says: Read and change your data on (some website)

Why it is needed   This permission is needed to read contents of the website you are about to crawl.

What the prompt says: Manage your downloads

Why it is needed   *New Since Chrome 65* there are some issues with Chrome blocking downloading XML file as potentially dangerous. This permission allows successfully downloading the sitemap.

What the prompt says: Read your browsing history

Why it is needed   This permission appears when extension is requesting ability to open and close tabs and access their contents. It does not read your browsing history. Please review project source code on Github if you need further verification.
